swift is an open-source pilot client which can allows users to connect their flight simulator to FSD-networks like VATSIM.

swift is an independent (private/non-commercial) software project creating open source software for flight simulaton.

Multiple simulator support!

swift supports major flight simulators like X-Plane 11, Prepar3D v1-v5, FSX, FS9 and Flightgear.

Support for other simulators can easily be added.

Multiple OS support!

swift is available for all major operating systems.

Open Source!

swift is the only pilotclient which is fully open-source under GPLv3. New contributors are always welcome!
