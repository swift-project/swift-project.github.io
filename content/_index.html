{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} swift is an open-source pilot client which can allows users to connect their flight simulator to FSD-networks like VATSIM. swift is an independent (private/non-commercial) software project creating open source software for flight simulaton. {{% /blocks/lead %}} {{< blocks/section color="secondary" >}} {{% blocks/feature icon="fa-plane-departure" title="Multiple simulator support!" %}} swift supports major flight simulators like X-Plane 11, Prepar3D v1-v5, FSX, FS9 and Flightgear. Support for other simulators can easily be added. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-desktop" title="Multiple OS support!" %}} swift is available for all major operating systems. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Open Source!" url="https://docs.swift-project.org/doku.php?id=getting_involved" %}} swift is the only pilotclient which is fully open-source under GPLv3. New contributors are always welcome! {{% /blocks/feature %}} {{< /blocks/section >}}
